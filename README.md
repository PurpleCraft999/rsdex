<h1>rsdex</h1>

A  little command line tool written in rust that acts like the pokedex

use
 `powershell -ExecutionPolicy Bypass -c "irm https://github.com/PurpleCraft999/rsdex/releases/download/v0.1.0/rsdex-installer.ps1 | iex" `
to install on windows and 
`curl --proto '=https' --tlsv1.2 -LsSf https://github.com/PurpleCraft999/rsdex/releases/download/v0.1.0/rsdex-installer.sh | sh`
to install on linux
